# Homepage Wireframe & Copy

## 1. Wireframe Description

### Header
- **Top Bar:** Language switcher (NL/EN), Phone number, "Request a Quote" button.
- **Main Navigation:** Logo, links to Product Catalogue, Calculator, About Us, News, Contact.

### Hero Section
- **Background:** High-quality image of reinforcement steel.
- **Content:** Large headline, subheading with value propositions, two CTA buttons.

### Featured Product Categories
- **Layout:** Grid of 3-4 category blocks.
- **Content per Block:** Image, Category Name (e.g., "Bouwstaalmatten"), short description, "View Products" button.

### Calculator Snippet
- **Layout:** Two-column section.
- **Left Column:** Brief explanation of the calculator.
- **Right Column:** Simplified input fields (e.g., floor area), "Calculate Now" button.

### Latest News/Blog Snippet
- **Layout:** Grid of 3 recent blog post previews.
- **Content per Preview:** Image, Title, short excerpt, "Read More" link.

### Footer
- **Layout:** Multi-column layout.
- **Columns:** Contact Info, Navigation Links, Company Info, Social Media Links.
- **Bottom Bar:** Copyright, links to Terms and Conditions, Privacy Policy.
